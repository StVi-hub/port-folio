POO Project

Conway's Game of Life in C++ is an application built using CLion and CMake. It features a graphical interface powered by SFML, custom save/load functionality, an editable grid, and multiple simulation modes for cellular automata dynamics. The project includes unit tests, a debug version, and a user-friendly interface for exploring the evolution of cellular automata. Originally devised by mathematician John Horton Conway in 1970, the Game of Life is a zero-player cellular automaton. Its evolution is determined by its initial state, allowing users to create configurations and observe their progression over time.
